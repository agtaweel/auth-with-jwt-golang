# auth-with-jwt-golang

Simple authentication project using JWT libarary for GO with MongoDB connection for database.

APIs list :
  -Signup
  -Login
  -Index Users
  -Show User
  -Add Product
  -Update Product
  -Index Products
  -Show Product
  -Delete Product


User Type:



  -Unsigned User:
    -Signup
    -Login
    
  -Signed User:
    -Show products
    -Show user
    
  -Admin User:
    -Index Users
    -Add Product
    -Update Product
    -Index Products
    -Delete Product
    
That's All, Only just a start into GoLang and MongoDB journey
